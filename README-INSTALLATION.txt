1. Place the Phase5_TeamAnger folder in the 'www' folder of the wamp/mamp server
2. Change files 'PHP/index.php' and 'PHP/connectStarTrekDB.php' variables or password '$pass' to system wamp/mamp password and variable for username, '$user' to wamp/mamp system username
3. Run the wamp/mamp server.
4. Go to URL 'http://localhost:8080/Phase5_TeamAnger/PHP/index.php' 
	This will setup the database and bring you to the first page in a way that doesn't recreate the database everytime you go to the homepage.
